  1. Hatching Technique

    2. Hatching involves using lines, either parallel or crossed, to create shading and texture in the sketch.

  2. Dussel Technique

    3. Similar to hatching, but this technique uses fine lines that are blended to create soft, gradual shading, adding depth to the sketch.

  3. Perspective Technique

    4. This technique uses the concept of perspective to draw objects from different viewpoints, creating the illusion of depth and space.

  4. Block Technique

    5. This involves filling in large areas of the drawing with a single color to represent the general shape of an object.

Functions of a Well-Processed Sketch from Various Sources:

  • 10 November, 2024
  • Sketch

  1. Quick Concept Development: Sketches allow for the rapid development of concepts without requiring detailed work. This is useful for quickly testing basic ideas.
  2. Basic Composition: A sketch provides a rough outline of how visual elements are arranged in a design. It is especially helpful in creating the basic structure.
  3. Visual Exploration: Sketches offer the freedom to experiment with shapes and visual styles, enabling boundless creativity.
  4. Sketches Depict Impressions and Mood: Sketches can capture the atmosphere or impression of the subject being depicted, focusing on the emotional expression or mood that is intended to be conveyed.
